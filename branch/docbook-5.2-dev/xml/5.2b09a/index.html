DocBook 5.2b09a

The most recent user documentation for this release is DocBook 5.1: The Definitive Guide.

This release consists of an NVDL file, a catalog, several schemas:

DocBook [RELAX NG, RNG Compact Syntax, Schematron]
The base DocBook 5.2b09a schema.
DocBook Assembly [RELAX NG, RNG Compact Syntax, Schematron]
The DocBook Assembly schema.
DocBook ITS [RELAX NG, RNG Compact Syntax, Schematron]
DocBook with the addition of the International Tag Set (ITS) elements and attributes.
DocBook XInclude [RELAX NG, RNG Compact Syntax, Schematron]
DocBook with the addition of XInclude elements in a variety of places; this allows for validation of pre-XIncluded documents.

and two tools designed to aid in transition for DocBook V4.x schemas:

db4-entities.pl
A Perl script that attempts to convert external parsed entities to XInclude elements.
db4-upgrade.xsl
An XSLT script that converts (most) DocBook V4.x schemas to DocBook V5.x.
